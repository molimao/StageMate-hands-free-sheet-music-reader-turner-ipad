## 👋 はじめに
**演奏を止めて、楽譜をめくる必要はもうありません。**

[StageMate](https://stagemate.site/) は、**「瞬き（ウィンク）」**や**「頭の傾き」**だけでページをめくることができる iOS アプリです。両手がふさがっているギター、ピアノ、バイオリンの演奏に最適です。

## ✨ 主な機能

* **👀 ハンズフリー譜めくり**: Bluetoothペダルは不要です。顔認識技術でスムーズにページを操作できます。
* **🎼 楽譜管理**: PDF、写真、画像をインポートして、すべてのスコアを一箇所に整理できます。
* **🎸 ライブ・パフォーマンス**: セットリスト作成機能、メトロノーム、常時画面オン機能など、ステージで役立つ機能が満載。
* **☁️ iCloud 同期**: iPhoneとiPadでライブラリを自動同期。

キーワード：*楽譜アプリ, 譜めくり, 電子楽譜, ギタータブ, ピアノ, バンドスコア, 自動スクロール*
