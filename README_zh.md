## 👋 简介
**别再让翻谱打断你的演奏了。**

[StageMate](https://stagemate.site/) 是一款专为吉他手、钢琴家和歌手设计的 iOS 应用。利用面部识别技术，你可以通过**眨眼**或**头部倾斜**来翻页，全程无需腾出手来触摸屏幕。

## ✨ 核心功能

* **👀 解放双手翻页**：识别眨眼动作或头部左右摆动，瞬间翻页。告别蓝牙踏板。
* **📂 统一乐谱管理**：支持导入 PDF、图片或相册照片。无论是吉他谱、五线谱还是简谱，统统收纳。
* **🎸 演出模式**：屏幕常亮，支持歌单（Setlist）排序，专为 Gig 设计。
* **☁️ iCloud 同步**：iPhone 和 iPad 乐谱库实时同步，排练演出无缝切换。

关键词：*电子乐谱, iPad 翻页器, 吉他谱管理, 钢琴谱阅读器, 眨眼翻页, 现场演出工具*
